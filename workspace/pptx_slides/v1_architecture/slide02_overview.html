System Architecture Overview

4-Layer Architecture

BioInsight AI is built on a modern, scalable architecture designed for biomedical research workflows. The system separates concerns across presentation, API, business logic, and data layers.

Key design principles: Modularity, Scalability, Real-time Processing, and AI-First approach.

Frontend Layer

React + Vite + TypeScript + Tailwind CSS

API Layer

FastAPI + REST + SSE Streaming

Core Features

Paper RAG, RNA-seq Pipeline, Knowledge Graph

Data Layer

ChromaDB + PostgreSQL + File Storage
