프로젝트 개요

프로젝트 목표

바이오/헬스케어 연구자를 위한 AI 기반 통합 연구 지원 플랫폼. 연구자의 발견을 가속화하되, 판단은 연구자에게.

핵심 기능

RNA-seq 6-Agent 파이프라인, ML 암종 예측, RAG 문헌 해석, 인터랙티브 HTML 리포트

활용 모델

Claude API (Opus/Sonnet), GPT-4o-mini, CatBoost + SHAP, PubMedBERT 임베딩

운영 환경

FastAPI + React + ChromaDB + DESeq2 (R) + Scanpy + PostgreSQL
