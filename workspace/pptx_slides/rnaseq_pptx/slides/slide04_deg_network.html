DEG Analysis & Network Construction

Agent 1: DEG Analysis

DESeq2 statistical analysis with apeglm log fold change shrinkage. Automatic filtering by adjusted p-value and fold change thresholds.

Supports both raw counts and pre-computed DEG tables. Handles batch effects with sample metadata integration.

Agent 2: Network Analysis

Spearman correlation matrix with vectorized numpy operations. Graph construction via NetworkX with adaptive hub centrality algorithm.

Dense graphs (>1M edges): weighted degree proxy. Large graphs (>1K nodes): k-sampling approximate betweenness.

Hub Gene Identification

Composite score = |log2FC| x -log10(padj). Top 20 hub genes selected as network centers.

Note: Hub genes reflect expression-level changes, NOT driver mutations (DNA-level).
