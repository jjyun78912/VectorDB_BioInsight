Q&A - RAG Pipeline

Q1: RAG와 일반 검색의 차이점은?

일반 키워드 검색은 정확한 단어 매칭만 가능. RAG는 의미적 유사도 기반 검색으로 "EGFR mutation" 검색 시 "epidermal growth factor receptor alteration"도 검색됨. LLM이 검색 결과를 종합하여 자연어 답변 생성.

Q2: PubMedBERT를 선택한 이유는?

일반 BERT는 Wikipedia, 뉴스 등 일반 텍스트로 학습. PubMedBERT는 PubMed 논문 전문으로 학습되어 바이오메디컬 용어 이해도가 높음. 바이오메디컬 벤치마크에서 10-15% 더 높은 정확도.

Q3: ChromaDB를 선택한 이유는?

세 가지 이유: (1) Python API가 간단 (2) 로컬에서 영구 저장 가능 (3) LangChain과 통합 용이. 대규모 프로덕션에서는 Pinecone, Weaviate 고려 가능.
