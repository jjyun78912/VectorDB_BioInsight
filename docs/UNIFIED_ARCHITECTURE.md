# BioInsight AI - Unified Platform Architecture

## Overview

**BioInsight AI**는 바이오·헬스케어 연구자를 위한 AI 기반 통합 연구 지원 플랫폼입니다.

> **"연구자의 발견을 가속화하되, 판단은 연구자에게"**
